Infrared Receiver pin position:
	"S" ====> Raspberry Pi pin 10
	"+" ====> Raspberry Pi +5V
	"-" ====> Raspberry Pi GND
	
Infrared Emitter pin position:
	"S" ====> Raspberry Pi 11
	"+" ====> Raspberry Pi +5V
	"-" ====> Raspberry Pi GND
	
LED pin position:
	"S" ====> Raspberry Pi 16(through resistor)
	"-" ===> Raspberry Pi GND
