AD0382 Pin position:
	CS  ===== Raspberry Pi Pin 11
	CLK ===== Raspberry Pi Pin 12
	DI  ===== Raspberry Pi Pin 13
	D0  ===== Raspberry Pi Pin 13
	CH0 ===== Hall Magnetic Sensor Pin "S" 
	VCC ===== Raspberry Pi +5V
	GND ===== Raspberry Pi GND

Hall magnetic pin position:
	"S" ==== ADC0382 Pin CH0
	"+" ==== Raspberry Pi +5V
	"-" ==== Raspberry Pi GND
	
LED pin position:
	"S" ==== Raspberry Pi Pin 16
	"-" ==== Raspberry Pi GND
