ADC0382 pin position:
	CS  =====  Raspberry Pi pin 11
	CLK =====  Raspberry Pi pin 12
	DO  =====  Raspberry Pi pin 13
	DI  =====  Raspberry Pi pin 13 
	CH0 =====  Small Sound Sensor pin A0
	VCC =====  Raspberry Pi +5V
	GND =====  Raspberry Pi GND

Small Sound Sensor pin position:
	D0  ====  Raspberry Pi pin 15
	A0  ====  ADC0382 Pin CH0
	"+" ====  Raspberry Pi +5V
	GND ====  Raspberry Pi GND
