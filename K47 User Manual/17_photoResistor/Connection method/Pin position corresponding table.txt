AD0382 Pin position:
	CS  ===== Raspberry Pi pin 11
	CLK ===== Raspberry Pi pin 12
	DI  ===== Raspberry Pi pin 13
	DO  ===== Raspberry Pi pin 13
	CH0 ===== Photo Resistor pin "S"
	VCC ===== Raspberry Pi +5V
	GND ===== Raspberry Pi GND

Photo Resistor pin position:
	"S" ==== ADC0382 pin CH0
	"+" ==== Raspberry Pi +5V
	"-" ==== Raspberry Pi GND
	
LED pin position:
	"S"  ==== Raspberry Pi pin 16
	"-"  ==== Raspberry Pi GND
